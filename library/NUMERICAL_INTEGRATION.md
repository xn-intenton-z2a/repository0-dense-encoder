Definition and relevance
- Numerical integration approximates continuous-time ODE solutions; Euler is simplest forward integrator.

Common integrators and choice rationale
- Euler (forward): first-order, simple, matched to discrete per-tick model.
- RK2/RK4: higher order but unnecessary for mission-specified per-tick discrete dynamics.

Error control and timestep selection
- For mission dt=1 tick is fixed; note potential for integration error but acceptable here.
