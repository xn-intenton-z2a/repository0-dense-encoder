# WEB_DEMO

Summary
Add a small interactive demo in src/web/ that uses the library to compute and display FizzBuzz results for a user-provided n.

Scope
- Minimal UI: numeric input, submit button, and result list.
- Use exported fizzBuzz function from src/lib/main.js to compute results and render them as a list with Fizz/Buzz highlighting.

Acceptance Criteria
- The demo is reachable via the repository website (npm start -> docs) or by opening the built docs and shows the demo UI.
- Entering 15 produces the expected 15-line result with 'FizzBuzz' on line 15.
- The demo uses the library API (not a separate implementation) so behaviour is consistent with unit tests.
